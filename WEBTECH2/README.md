# Student Profile Web Page 🎓

A responsive, vibrant, and feature-rich Student Profile dashboard built using **HTML5**, **Bootstrap 5**, and custom **CSS3**. This project presents clean academic visualization metrics alongside skill bars and official certifications.


## ✨ Features & Project Structure

The project strictly fulfills all requirements using proper semantic HTML structure and micro-interaction styling:

### 1. Navigation Header
* **Responsive Navbar:** Features standard links for **Home**, **About**, and **Contact** with slick hover transitions.
* **Social Profiles:** Includes inline modern icon anchors (Twitter, Bookmarks, Globe) powered by Font Awesome.

### 2. Left Panel: Profile & Technical Skills
* **Profile Wrapper:** Features student photo (`profile.jpeg`) wrapped in a dynamic **cyan-to-blue gradient ring indicator**.
* **Personal Data:** Accurately renders student profile metrics:
  * **Name:** Swarna Sahithya
  * **Roll Number:** A24126511049
  * **College:** ANITS Engineering College & Sciences
  * **Branch:** Information Technology
* **Interactive Skill Grid:** Displays skill blocks for **HTML5**, **CSS3**, **Python**, and **Java** outfitted with specific brand colors and progress level indicators.
* **Call to Action:** Includes a custom styled dynamic `View Profile` button layout.

### 3. Right Panel: Academics & Accreditations
* **Academic Ledger Matrix:** A custom, hoverable table structured exactly as requested:
  * **Subject Name** acts as the primary layout row identifier.
  * Displays distinct columns for **Internal Marks**, **External Marks**, and **Total Marks** metrics.
* **Notes & Certifications Tracker:** Dedicated list element tracking technical upskilling achievements:
  * Google Cloud (Introduction to Generative AI)
  * TCS iON (Career Edge - Young Professional)
  * Qubitech.Wiser (Quantum Computing)

## 🛠️ Technologies Used

* **HTML5:** Structuring layouts cleanly using semantic components (`<main>`, `<aside>`, `<section>`).
* **CSS3:** Custom linear gradients, responsive typography tweaks, custom layouts, and vector pattern backgrounds.
* **Bootstrap 5 (via CDN):** Handled primary grid containers, flex alignments, mobile menu toggles, and typography baselines.
* **Font Awesome (v6):** Icon assets used in navigation link elements and language badges.

## 💻 How to Run This Project Locally

1. Clone or download this repository directory to your local device.
2. Open the project folder inside **VS Code**.
3. Make sure you have your profile image saved inside the directory named exactly as **`profile.jpeg`**.
4. Right-click on `index.html` and select **"Open with Live Server"**.
